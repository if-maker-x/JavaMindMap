# Java 架构思维导图工作台

这是一个本地可运行的网页前后端工具，用来读取指定 Java 源码目录，自动归纳出按 package 和 class 组织的架构思维导图，并允许你继续在图上编辑、补充和保存，用于后续规划与迭代。

## 现在支持的能力

- 输入本地路径分析 Java 目录
- 支持带双引号的路径输入
- 手动选择文件夹后直接分析其中的 `.java` 文件
- 自动抽取 package、class / interface / enum / record
- 抽取字段、构造器、方法、继承、实现和项目内依赖
- 画布支持拖动画布、拖动节点、滚轮缩放、适应视图
- 通过多选开关控制画布中是否显示字段、方法、继承、依赖等结构
- 右侧详情面板始终展示完整类信息，便于先总览再下钻
- 保存当前图谱并重新加载

## 启动

```bash
node server.js
```

启动后访问：

`http://localhost:3000`

## 使用建议

1. 先通过“从路径分析”或“选择文件夹”生成第一版图谱。
2. 先关闭字段 / 方法等细节开关，看全局模块结构。
3. 聚焦某个 package 或 class 后，再打开字段、构造器、方法等细节。
4. 在右侧备注区记录当前版本规划、重构思路和下一步任务。
5. 调整好后保存图谱，作为后续迭代基线。

## 当前实现说明

- 后端：原生 Node.js `http`
- 前端：原生 HTML / CSS / JavaScript
- Java 解析：轻量规则提取，适合架构梳理和规划，不是完整 AST 级解析器
- 保存位置：`data/saved-map.json`
